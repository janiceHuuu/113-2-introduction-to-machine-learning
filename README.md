- 作業一：運用水泥構件強度資料實作多變數線性迴歸模型，並分析其屬於監督式或非監督式學習

- 作業二：實作並探討不同深度的FCN分類準確度

- 作業五：運用Learning rate schedule、Go deep等方法，增加Residual network的分類準確度，並比較VGG和Residual network於分類問題之準確度
